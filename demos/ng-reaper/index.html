Confidence tiers

No scores, no ML — every tier is explainable

SAFE

No references found anywhere in the workspace. Auto-deletable with --fix — the only tier that is.

LIKELY

No static references, but a pattern exists that could resolve dynamically (e.g. providedIn:'root' + possible Injector.get). Flagged, never auto-fixed.

SUSPECT

Referenced only through a dynamic pattern ng-reaper can't fully verify. Surfaced for a human to check, never touched automatically.
